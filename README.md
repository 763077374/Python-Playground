#性能指标<br>
>常数级O(1)<br>
>对数级O(log n) <br>
>次线性级O(n^d),其中d<1<br>
>线性级O(n)<br>
>线性对数级 O(nlog n)<br>
>平方级 O(n^2) <br>
>指数级 O(2^n)
